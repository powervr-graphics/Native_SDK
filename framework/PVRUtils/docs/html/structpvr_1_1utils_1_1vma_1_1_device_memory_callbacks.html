Detailed Description

The DeviceMemoryCallbacks struct defines a set of callbacks that the library will call for vkAllocateMemory and vkFreeMemory. Provided for informative purpose, e.g. to gather statistics about number of allocations or total amount of memory allocated in Vulkan. Used in AllocatorCreateInfo::pDeviceMemoryCallbacks.
