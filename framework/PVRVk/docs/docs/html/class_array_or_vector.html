Detailed Description

template<typename ElementType, size_t static_size>
class ArrayOrVector< ElementType, static_size >

Internal helper class that is used for unknown size arrays, utilising a small statically allocated space to avoid dynamic allocations if an array of "less-than" a number of items is required

Template Parameters
ElementTypeThe type of items stored
static_sizeThe amount of statically allocated memory used. If overran, a dynamic allocation takes place.
