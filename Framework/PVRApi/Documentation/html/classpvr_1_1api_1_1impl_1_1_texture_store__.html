Detailed Description

The class powering any texture. Wraps the underlying API Texture object and represents the storage bits of any texture. Is normally used through a TextureView object. Wrapped and accessed in a TextureStore Reference Counted Framework object.

Allocate a transient 2D texture. A transient texture is one that will be used "logistically" between subpasses/draws but the implementation is encouraged to NOT allocate memory for. It is invalid to call this function on an already allocated texture.

Parameters
formatTexture format
widthTexture width
usageBitwise-or'ed allowed usage flags for this texture
imageLayoutInitial layout to which the image will be created

A typical use for a transient attachment, is a G-Buffer.
