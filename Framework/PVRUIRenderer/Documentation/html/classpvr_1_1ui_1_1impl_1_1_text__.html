Text_ & pvr::ui::impl::Text_::setText ( const std::string &  str)

Set the text of this sprite.

You must always submit your outstanding operations to a texture before calling setText. Because setText will edit the content of VBOs and similar, these must be submitted before changing the text. To avoid that, prefer using more Text objects.
