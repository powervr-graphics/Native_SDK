Detailed Description

Include this file if you wish to use the PVRShell functionality, but you will NOT be linking with a PVRApi library. Only include this file once, in one of your CPP files. If you need to include PVRShell in multiple locations, include this file in one of your main cpp files, and PVRShell.h in the rest.
